# 题库知识点

分类/回归：x->y的映射，分类问题y为离散值，回归问题y为连续值

分类强调依据类别标签 y 对样本 x 空间的划分，回归强调 x 与回归值的拟合。

聚类：学习 x 到类簇 y 的映射

"类簇"通常指的是一组相似的数据点的集合。这种相似性通常是通过某种度量（例如欧几里得距离或余弦相似性）来确定的。
